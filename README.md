# Girls

1. Меняете имя ноды в apps/vladlena/vladlena.ex:5 на `vladlena@computer_name`
2. Меняете имя ноды в apps/elena/elena.ex:5 `elena@computer_name`

4. Вызываете `Vladlena.send_transactions` в окне у `vladlena`
